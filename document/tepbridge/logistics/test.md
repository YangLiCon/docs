# 报修管理系统

这是一个简单的报修管理系统，具有以下功能：

1. 部门管理
2. 用户管理（包括报修人员和接修人员）
3. 报修管理
4. 费用记录（材料费用和人工费用）
5. 按月统计分析

## 使用方法

1. 直接在浏览器中打开 `client/index.html` 文件即可运行系统
2. 首次使用时，需要先添加部门，然后添加用户
3. 用户分为两种角色：报修人员和接修人员
4. 报修人员可以创建报修单
5. 接修人员可以处理报修单并记录费用
6. 系统可以按月生成统计报表

## 功能详情

### 1. 部门管理
- 添加、编辑、删除部门

### 2. 用户管理
- 添加、编辑、删除用户
- 分配用户到部门
- 设置用户角色（报修人员/接修人员）

### 3. 报修管理
- 创建报修单
- 分配接修人员
- 更新报修状态
- 记录完成日期和费用

### 4. 统计分析
- 按部门统计报修数量和费用
- 按接修人员统计处理数量和费用
- 总费用统计

## 注意事项

1. 本系统使用浏览器本地存储，刷新页面后数据会丢失
2. 如需持久化存储，可以考虑后端开发或使用本地存储API
3. 为获得最佳体验，请使用现代浏览器
